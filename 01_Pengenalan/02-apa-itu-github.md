## ğŸ’¡ Pembahasan Inti

### GitHub: Rumah Bagi Kode Kamu
Git memang hebat dalam mencatat sejarah perubahan di komputermu secara lokal. Namun, bagaimana jika laptopmu rusak? Atau bagaimana jika kamu ingin mengajak temanmu di UNRI untuk ikut ngoding bareng?

Di sinilah **GitHub** berperan. GitHub adalah layanan berbasis *cloud* yang digunakan untuk menyimpan **Repository** Git secara online. GitHub memudahkan kita untuk berbagi kode, berkolaborasi dengan tim dari mana saja, dan berkontribusi ke proyek *open-source*.

### Kenapa Bukan Google Drive atau Dropbox?
Mungkin kamu berpikir, "Kan bisa taruh folder kodingan di Google Drive?". Memang bisa, tapi GitHub punya fitur khusus yang tidak dimiliki cloud storage biasa:
1. **Code Preview:** GitHub bisa menampilkan baris kode dengan warna yang rapi (*syntax highlighting*).
2. **Issue Tracking:** Tempat diskusi jika ada *bug* atau fitur yang ingin ditambahkan.
3. **Pull Requests:** Fitur paling sakti untuk meninjau kodingan teman sebelum digabungkan ke proyek utama.
4. **GitHub Actions:** Automasi untuk mengetes kodinganmu secara otomatis.

Bayangkan GitHub itu seperti **Instagram**, tapi isinya bukan foto makanan atau selfie:

* **Profile:** Adalah CV atau portofolio kamu. Orang bisa melihat seberapa aktif kamu ngoding.
* **Repository:** Adalah postingan atau album proyekmu.
* **Star:** Sama seperti **Like**, sebagai bentuk apresiasi orang lain terhadap karyamu.
* **Fork:** Seperti **Share/Retweet**, kamu mengambil karya orang lain untuk kamu simpan dan modifikasi di akunmu sendiri.
* **Follow:** Kamu mengikuti perkembangan developer hebat lainnya.

## â“ Pertanyaan Umum (Q&A)

**T: Apakah GitHub berbayar?**
**J:** Untuk penggunaan pribadi dan tim kecil, GitHub sepenuhnya **GRATIS**. Kamu bisa membuat repository sebanyak mungkin tanpa biaya.

**T: Apakah kodingan saya bisa dilihat orang lain?**
**J:** Kamu bisa memilih. **Public** (siapa saja bisa lihat) atau **Private** (hanya kamu dan orang yang kamu undang yang bisa lihat).

**T: Apa ada selain GitHub?**
**J:** Ada! Ada **GitLab** dan **Bitbucket**. Tapi saat ini GitHub adalah yang paling populer di dunia industri.
